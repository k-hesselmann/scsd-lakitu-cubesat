Six Flight Phases

Phase 1
Standby
System powered on, sitting on ground. All sensors initialising. Ground pressure baseline recorded here as reference for all altitude calculations.
IMU Baro GPS
Phase 2
Launch
Balloon released. Sharp upward jerk, brief chaotic motion. Lasts seconds. System transitioning from ground-constrained to free flight.
IMU ★ Baro
Phase 3
Ascent
Balloon stabilised, climbing at 3 to 5 m/s steadily. Longest active phase, typically 1.5 to 2 hours. Temperature dropping significantly with altitude.
Baro ★ GPS
Phase 4
Cruise
Balloon fully expanded at float altitude (~30 km). Vertical velocity near zero, altitude oscillating within a bounded range. Duration 30 min to a few hours.
GPS ★ Baro
Phase 5
Descent
Balloon burst. Rapid fall begins. IMU sees sudden shift from near 0g float to ~1g. Parachute deploys, stabilising descent rate.
IMU ★ GPS Baro
Phase 6
Landing
System touches down. All motion stops. Terminal state — no further transitions. Logging and telemetry continue for recovery. Shutdown timer starts.
GPS ★ Baro

Cross-Cutting Rules

  • Median filter with window of 5 samples applied to all sensor parameters before any threshold comparison.
  • All persistence windows count consecutive filtered samples, not raw samples.
  • GPS fix status checked before every GPS parameter use. If fix invalid, fallback applies automatically.
  • GPS must be configured in Airborne mode above 18 km. CDH handles configuration. FSW monitors fix status.
  • Ground pressure baseline recorded and stored in Standby. All barometric altitude calculations relative to this value.
  • IMU float baseline tracked as running median during Cruise for relative change detection at burst.
  • Every state transition persisted to non-volatile memory within 1 second of transition.
  • Watchdog timer serviced within every 10 second window in all phases.
  • No invalid or undefined state transitions permitted. FSM only allows the 5 defined paths.
  • Direct Ascent to Descent transition permitted if Cruise is never detected (balloon burst before float).
  • Landing shutdown timer starts on Landing entry. Non-essential systems powered down after defined period.
